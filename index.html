About Staticky

Staticky is a simple static site builder that uses a set of layouts and configuration variables to quickly populate page layouts with content.

Creating Layouts

written 2014-07-29

The site's layout files (in the layouts folder) are what determine how the site is formatted and where to place the post content. The generator is designed to create an index.html page with 5 post thumbnails, a page for all post thumbnails, and individual linkable pages for each post.

Using the Python File

written 2014-07-29

The python file has three main commands:
1. generate: assembles layouts to create the site.
2. new_post: creates a new post with the proper header in the posts folder.
3. preview: constantly re-generates the site to allow you to view changes in the web browser as you edit.

In order to run staticky you must have python 3.0+ installed on your computer.

Configuration File

written 2014-07-28

The config.txt file that is included with staticky allows for the creation of variables for populating the site. Each value stored in the config can be used to replace a value surrounded by double square brackets in the layout. For example, the site "title" value in the config file will replace anywhere "[[title]]" is placed in the site's layout files.
