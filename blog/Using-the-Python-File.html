Using the Python File

written 2014-07-29

The python file has three main commands:
1. generate: assembles layouts to create the site.
2. new_post: creates a new post with the proper header in the posts folder.
3. preview: constantly re-generates the site to allow you to view changes in the web browser as you edit.

In order to run staticky you must have python 3.0+ installed on your computer.

The staticky.py file is open source and easily editable using any text editor. Feel free to make changes for your specific site.
